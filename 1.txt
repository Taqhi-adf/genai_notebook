# What is Generative AI (GenAI)?

Generative AI (GenAI) is a type of Artificial Intelligence that can create new content like:

* text
* images
* videos
* music
* code
* chat responses

by learning from large amounts of data.

---

# Simple Meaning

Normal AI:

* analyzes data
* predicts answers

GenAI:

* creates new things

Example:

If you ask:

```text id="w8yivw"
Write a poem about cricket
```

GenAI can generate a completely new poem.

# Simple Python Example

Using Google Gemini:

```python id="jlwmk5"
import google.generativeai as genai

genai.configure(api_key="YOUR_API_KEY")

model = genai.GenerativeModel("gemini-2.0-flash")

response = model.generate_content(
    "What is AI?"
)

print(response.text)
```

# How to Upload Huge Files (>25MB) to GitHub
GitHub normally blocks files larger than:

* 25 MB warning
* 100 MB hard limit

For large AI/ML files use:

* Git LFS (Large File Storage)

---

# Best Method: Git LFS

Git LFS is used for:

* models
* datasets
* PDFs
* videos
* checkpoints
* zip files

---

# Step-by-Step Process

---

# Step 1: Install Git LFS

## Windows

Download:

[Git LFS Download](https://git-lfs.com?utm_source=chatgpt.com)

Install normally.

---

## Verify Installation

Open terminal:

```bash id="u8hzjg"
git lfs install
```

Expected:

```text id="o0l5zk"
Git LFS initialized.
```

---

# Step 2: Clone Repository

```bash id="2ljlwm"
git clone https://github.com/USERNAME/REPO.git
```

Example:

```bash id="jlwmm1"
git clone https://github.com/rahul/ai-project.git
```

---

# Step 3: Move into Repository

```bash id="jlwmm2"
cd ai-project
```

---

# Step 4: Track Large Files

Example for ZIP files:

```bash id="jlwmm3"
git lfs track "*.zip"
```

Example for model files:

```bash id="jlwmm4"
git lfs track "*.bin"
```

Example for PDFs:

```bash id="jlwmm5"
git lfs track "*.pdf"
```

---

# Step 5: Check `.gitattributes`

Git automatically creates:

```text id="jlwmm6"
.gitattributes
```

Example content:

```text id="jlwmm7"
*.zip filter=lfs diff=lfs merge=lfs -text
```

---

# Step 6: Add Huge File

Example:

```bash id="jlwmm8"

```

OR

```bash id="jlwmm9"

```

---

# Step 7: Commit

```bash id="jlwmma"

```

# Step 8: Push to GitHub

